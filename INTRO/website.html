HTML Basics: Tags and Lists

Learn about basic HTML tags and how to create lists on your webpage.

HTML Tags

HTML tags are the building blocks of a webpage.

<h1> Heading Tag </h1> <p> This is a paragraph. </p>
Paragraph Tags

Use the <p> tag to add paragraphs to your content.

<p>This is a paragraph.</p>
Lists in HTML

Learn how to create lists in HTML.

  • Ordered Lists
  • Unordered Lists

Types of Lists

Ordered List

An ordered list uses numbers:

<ol> <li>First item</li> <li>Second item</li> <li>Third item</li> </ol>
Unordered List

An unordered list uses bullet points:

<ul> <li>Apple</li> <li>Banana</li> <li>Cherry</li> </ul>
